### Featured Projects

#### [AlgoChef (SaaS)](https://github.com/omarcasey/AlgoChef)
Advanced trading analytics platform built with Next.js 13 & TypeScript
- Comprehensive SaaS solution for analyzing trading strategies across any brokerage platform
- Features advanced quantitative analysis including Sharpe ratio, Monte Carlo simulations, and VaR
- Proprietary portfolio optimization algorithm using modern portfolio theory
- Intuitive data import system with custom column mapping
- Interactive data visualizations and real-time analytics dashboard
- 100+ monthly active users

#### [Sqribe.ai (SaaS)](https://github.com/omarcasey/sqribe_app)
AI-powered video dubbing platform built with Next.js 13 & Firebase
- Enables content creators to dub videos into multiple languages while preserving their original voice
- Scalable cloud infrastructure using Google Cloud Functions and Pub/Sub for distributed processing
- High-fidelity voice cloning system with ElevenLabs API achieving 95% voice similarity
- Asynchronous video processing pipeline with RESTful APIs for service orchestration
- Intuitive video editing interface for non-technical users
- 80+ monthly active users
